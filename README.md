## ComfyUI Video Workflow Extractor

ComfyUI Video Workflow Extractor allows you to extract the ComfyUI workflow from a video file.

**All processing is done in the browser. No data is sent to any server.**

## How to use

1. Drag and drop a .mp4 video file into the application.
2. The application will analyze the video file and extract the workflow.
3. You can then copy the workflow to your clipboard.

[Try it out here!](https://gabecastello.github.io/comfyui-video-workflow-viewer/)
